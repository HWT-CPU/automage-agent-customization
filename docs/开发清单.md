## 7. 节点 Cron 时间表速查

| 节点 | 时间 | 动作 |
|---|---|---|
| Staff Agent | 每日 18:00 | 飞书群发日报卡片 |
| Staff Agent | 每日 20:00 | 二次催填未提交者 |
| Manager Agent | 每日 21:00 | 读取本部门日报 → 生成汇总报表 |
| Manager Agent | 每日 22:00（待确认） | 若仍有 ≥20% 未交，告警 *"信息不全预警"* |
| Executive Agent | 由 Dream 触发 | 决策提案 / 任务拆解 / 异常报警 |

---

## 8. 接口契约速查

| 接口名 | 方法 | 路径 | 调用方 | 用途 |
|---|---|---|---|---|
| 身份初始化 | POST | `/api/v1/agent/init` | 所有 Agent | 上报 `node_id` 获取权限 |
| 日报提交 | POST | `/api/v1/report/staff` | 执行节点 | 写入校验过的员工日报 |
| 汇总上报 | POST | `/api/v1/report/manager` | 中枢节点 | 写入汇总后的部门动态 |
| 指令下发 | POST | `/api/v1/decision/commit` | 决策节点 | 老板按钮后下发到 `task_queue` |
| 任务获取 | GET  | `/api/v1/tasks` | 全员 | 按角色拉未完成任务 |

> 统一响应格式：`{ "code": 200, "data": {}, "msg": "success" }`

---

## 9. 异常处理基线

- [ ] **422 数据格式错误**：根据返回提示自我修正后重试，最多 N 次（待定）。
- [ ] **接口超时**：指数退避 1s → 2s → 4s → 8s → 16s。
- [ ] **鉴权失败**：定期轮询 `auth_status`，失效自动推送管理员 IM。
