# 后端启动说明
日期：2026-05-07
项目代码地址：<https://github.com/HWT-CPU/automage-agent-customization>

## 一、启动前准备

请先确认本机已安装：

- Python
- PostgreSQL

并确保项目目录为：

```powershell
D:\Code\A实习项目\automage-agent-customization-main
```

## 二、连接远程数据库

当前启动后端时，直接连接远程数据库即可，不需要先执行本地初始化脚本。

当前项目默认远程数据库配置已写在：

- `configs/automage.local.toml`

远程库连接信息：

- host：`182.92.93.16`
- port：`5432`
- database：`automage`
- user：`automage`

## 三、启动后端服务

然后执行：

```powershell
python scripts\run_api.py
```

作用：

- 启动 FastAPI 后端服务
- 默认监听 `0.0.0.0:8000`
- 提供 Swagger、OpenAPI 和健康检查接口

## 四、启动后访问地址

启动成功后，可直接访问：

- Swagger：`http://localhost:8000/docs`
- OpenAPI：`http://localhost:8000/openapi.json`
- 健康检查：`http://localhost:8000/healthz`

## 五、启动顺序

推荐顺序如下：

1. 确认远程数据库可连通
2. `python scripts\run_api.py`
3. 打开 `http://localhost:8000/docs`

## 六、补充说明

- 远程数据库已经可用时，不需要再执行 `init_db.py`。
- `run_api.py` 是启动服务。
- 如果后续需要改表结构，再单独处理数据库迁移，不要在日常启动里重复初始化。
